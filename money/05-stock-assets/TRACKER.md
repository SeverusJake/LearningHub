## Weekly review

Answer these 5 questions every week, in this file, using that week's rows above:

1. **Assets submitted this week?** (Total count uploaded, across all niches.)
2. **Assets accepted this week?** (Count approved into the live catalog — pull this from the contributor portal's review status.)
3. **Assets rejected this week, and why?** (Count plus the actual rejection reasons given — technical, similar content, compositional, etc. — this is the data that should drive next week's prompts.)
4. **Downloads this week?** (Total downloads across the live catalog, per the contributor portal's stats.)
5. **Revenue this week, and which niche is performing best right now?** (State the dollar total, name the specific niche with the best acceptance-rate-times-downloads combination, and decide whether next week's session doubles down on that niche or keeps testing a new one.)
