# Game 10 — Capstone: Pixel Quest

**Track:** gamedev · **Engine:** Godot 4 (+ HTML5 export) · **Platform:** Steam (paid) + web demo · **Monetization:** premium + demo-to-wishlist · **Difficulty:** 💀💀💀💀💀 · **Time:** 4-6 weeks · **Prerequisites:** Games 01-09

## Concept

Pixel Quest is the capstone: a small but genuinely *complete* commercial 2D game — one focused genre (an action-platformer or a top-down adventure) with about 30-60 minutes of real content — released for real on Steam, with a free HTML5 web demo that funnels players to the Steam page and drives wishlists. This is the graduation piece. Every earlier game taught one slice — a mechanic, an engine, a platform, a monetization model. Here you assemble the whole thing to a shippable, sellable standard: a finished game with a save system, an options menu, controller support, real art, a Steam store presence, and a launch. It's the portfolio centerpiece you point people at.

## AI does / What you do

**AI does:** the full Godot game (structure, mechanics, save system, options, controller support), the HTML5 export config, the Steam integration, and the store-page copy draft — the bulk of the build.

**You do:** the creative direction and scope calls, creating/verifying the Steamworks account, tax/bank paperwork, paying the $100 Steam Direct fee, generating all final art and music, hosting the web demo, running the SteamPipe build and store submission, and the launch itself.

## Deliverables

- [ ] A complete, beatable 2D game in Godot 4 with ~30-60 min of content, a save system, an options menu, and controller support
- [ ] A polished vertical slice exported to HTML5 as a free, hosted web demo that funnels to the Steam page
- [ ] A real Steam release: store page with real capsules/trailer/screenshots, wishlists, the $100 Direct fee paid, tax/bank done, a SteamPipe build, and achievements
- [ ] Real (AI-generated) final assets throughout — no placeholders shipped
- [ ] All 20 acceptance-checklist items in GUIDE.md passed
